消失之物题解

Posted on 2021-11-11，3 min read

这道题需要一点小小的思维转弯，然而我太死板了转了好久才转过来……

一个非常显然但是非常关键的式子：不用第 iii 的物品凑出体积为 jjj 的方案 = 所有凑出体积为 jjj 的方案 - 使用了 iii 物品的方案。

重点是怎么去求使用了 iii 物品凑出体积为 jjj 的方案，考虑到我们一定使用了 iii 物品，我们可以先假设去掉 iii 的体积，去掉后就变成了不使用 iii 凑成体积为 j−cij - c_ij−ci​ 的方案。

嗯？这不就是题设的 cnti,j−cicnt_{i , j - c_i}cnti,j−ci​​ 吗？因为 j−cij - c_ij−ci​ 一定会变小，所以可以形成递推关系，所以有：cnti,j=fj−cnti,j−cicnt_{i , j} = f_j - cnt_{i , j - c_i}cnti,j​=fj​−cnti,j−ci​​，fjf_jfj​ 表示所有凑出体积为 jjj 的方案。

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 2021;
const int mod = 10;

int n , m , f[N] , c[N] , cnt[N][N];

int main(void){

    scanf("%d%d" , &n , &m);

    for(int i = 1 ; i <= n ; ++ i){
        scanf("%d" , &c[i]);
    }

    f[0] = 1;

    for(int i = 1 ; i <= n ; ++ i){
        for(int j = m ; j >= c[i] ; -- j){
            f[j] = (f[j] + f[j - c[i]]) % mod;
        }
    }

    for(int i = 1 ; i <= n ; ++ i){
        cnt[i][0] = 1;
    }

    for(int i = 1 ; i <= n ; ++ i){
        for(int j = 1 ; j <= m ; ++ j){
            if(j < c[i]){
                cnt[i][j] = f[j] % mod;
            }
            else cnt[i][j] = (f[j] - cnt[i][j - c[i]] + mod) % mod;
            printf("%d" , cnt[i][j]);
        }
        putchar('\n');
    }

    return 0;

}

总的来说，这道题是一个小小的容斥，解决这道题非常关键的一点就是能不能想到把使用了 iii 物品的方案转化为去掉后就变成了不使用 iii 凑成体积为 j−cij - c_ij−ci​ 的方案。
