奶牛会展题解

Posted on 2021-11-12，3 min read

这道题刚开始我想的是设 fa,b=0/1f_{a , b} = 0/1fa,b​=0/1 表示智商为 aaa，情商为 bbb 的状态能否被凑出来，但是这样显然时间空间都是过不去的

后来想到设 fi,af_{i , a}fi,a​ 表示所有只考虑前 iii 头牛，智商为 aaa ，情商的最大值，因为如果把智商放到了状态里，情商肯定越大越好

但是有一个问题我没能解决，就是虽然说题目要求智商情商都不为 0 ，但是选牛的过程中是可以为负数的，因为题设要求的是最终选出来的牛智商情商都不为 0。

一开始我想的是吧所有数都加上一个定值 kkk 使得所有数都是非负数，但是又有一个问题就是我不知道我选了几头牛也就是说我不知道答案加了几个 kkk ，不知道原来的答案是多少了，所以还得在状态里设一个 cntcntcnt 表示选了几头牛，但是时间空间都已经承受不住了。

又想到了将智商都加上一个 kkk ，情商都减去一个 kkk 但是还是不知道情商减了几个 kkk。

实在没有办法，打开了题解。

其实只需要把数组整体右移就可以了，这里因为是将智商整体加上一个 kkk 所以就不需要再设 cntcntcnt 计算多加了多少，只需要 −=k-=k−=k 就可以了

还有一点需要注意，这道题因为智商值很大，再加上 iii （表示只考虑前 iii 头奶牛） 就会炸空间，如果不开的话，不满足无后效性，所以考虑对正数和负数分别处理，正数照常倒序枚举，负数只需要反一下就可以

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 410;
const int M = 8e5 + 10;

int n , f[M];

struct node{
    int w , v;
}a[N];

int main(void){

    scanf("%d" , &n);

    for(int i = 1 ; i <= n ; ++ i){
        scanf("%d%d" , &a[i].w , &a[i].v);
    }   

    memset(f , ~0x3f , sizeof(f));

    f[400000] = 0;

    for(int i = 1 ; i <= n ; ++ i){
        if(a[i].w >= 0){
            for(int j = 800000 ; j >= a[i].w ; -- j){
                f[j] = max(f[j] , f[j - a[i].w] + a[i].v);
            }
        }
        else{
            for(int j = 0 ; j <= 800000 + a[i].w ; ++ j){
                f[j] = max(f[j] , f[j - a[i].w] + a[i].v);
            }
        }
    }

    int ans = 0;

    for(int i = 400000 ; i <= 800000 ; ++ i){
        if(f[i] >= 0){
            ans = max(ans , f[i] + i - 400000);
        }
    }

    printf("%d" , ans);

    return 0;

}

处理正数时，看到题解里写 j>=a[i].wj >= a[i].wj>=a[i].w ，愣了一下，仔细想一下，对的，因为中途可能出现负数
