绝世好题题解

2021-12-09

这道题自己做出来了，发现比 191919 年的自己长进非常大

首先，设 fif_ifi​ 表示所有以 iii 为结尾的序列的最长长度，按照最长上升子序列的求法，考虑 iii 前一个是那个数。因此，有

fi=max(fi,fj+1)，j∈[1,i−1]f_i = max(f_i , f_j + 1)，j∈[1 , i - 1] fi​=max(fi​,fj​+1)，j∈[1,i−1]

其中需要满足 aia_iai​ & aja_jaj​

然后我们发现复杂度较大，考虑优化

我们考虑将 aia_iai​ 写成二进制，发现，只要 aja_jaj​ 拥有 aia_iai​ 的任何一个 111 的二进制位，那么就可以转移。对于aia_iai​ 的每一个 111 的二进制，如果 aja_jaj​ 也有这个二进制位，那么显然 jjj 可以转移。因此，我们考虑快速求出前 i−1i - 1i−1 个 aja_jaj​ 有这个二进制位的 fjf_jfj​ 的最大值

设 gi,kg_{i , k}gi,k​ 表示前 iii 个 aaa 值中有 (1<<k)(1 << k)(1<<k) 这个二进制位的最大 dpdpdp 值，因此，有：

gi,k=max(gi−1,k,fi)g_{i , k} = max(g_{i - 1 , k} , f_i) gi,k​=max(gi−1,k​,fi​)

其中 iii & (1<<k)(1 << k)(1<<k)

还需要注意，如果 !i! i!i & (1<<k)(1 << k)(1<<k)，gi,k=gi−1,kg_{i , k} = g_{i - 1 , k}gi,k​=gi−1,k​，因为此时 fif_ifi​ 更新不到 tatata 了，但是，ggg 又是一个前缀 maxmaxmax 数组，所以需要将上一次的承接过来

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 1e5 + 10;
const int M = 35;

int n , a[N] , f[N] , g[N][M] , ans;

int main(void){

    scanf("%d" , &n);

    for(int i = 1 ; i <= n ; ++ i){
        scanf("%d" , &a[i]);
        f[i] = 1;
    }

    for(int i = 1 ; i <= n ; ++ i){
        for(int k = 0 ; k <= 30 ; ++ k){
            if((1 << k) & a[i]){
                f[i] = max(f[i] , g[i - 1][k] + 1);
            } 
        }
        for(int k = 0 ; k <= 30 ; ++ k){
            if((1 << k) & a[i]){
                g[i][k] = max(g[i - 1][k] , f[i]);
            }
            else g[i][k] = g[i - 1][k];
        }
        ans = max(ans , f[i]);
    }

    printf("%d" , ans);

    return 0;
}
