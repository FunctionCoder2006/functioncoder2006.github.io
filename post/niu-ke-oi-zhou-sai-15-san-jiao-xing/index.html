牛客OI周赛15三角形

2021-11-11 #背包 #动态规划

通过数据范围可以发现，最大能凑出来的价值不超过 1e41e41e4

因此考虑 dpdpdp 设 fi,jf_{i , j}fi,j​ 表示只考虑前 iii 个宝盒，凑出价值为 jjj 的方案，这里需要注意，因为每个物品都必须选，所以不能写 f[i][j]=f[i−1][j]f[i][j] = f[i - 1][j]f[i][j]=f[i−1][j]

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 110;
const int M = 1e4 + 10;

int n , k , m[N] , w[N];
int f[N][M];

int main(void){

    scanf("%d%d" , &n , &k);

    f[0][0] = 1;

    for(int i = 1 ; i <= n ; ++ i){
        scanf("%d" , &m[i]);
        for(int j = 1 ; j <= m[i] ; ++ j){
            scanf("%d" , &w[j]);
        }
        for(int j = 10000 ; j >= 0 ; -- j){
            for(int k = 1 ; k <= m[i] ; ++ k){
                if(j >= w[k])f[i][j] += f[i - 1][j - w[k]];
            }
        }
    }

    int cnt = k;
    long long ans = 0;

    for(int j = 1 ; j <= 10000 ; ++ j){
        if(f[n][j]){
            if(cnt >= f[n][j]){
                ans += 1ll * j * f[n][j];
                cnt -= f[n][j];
            }
            else {
                ans += 1ll * cnt * j;
                break;
            }
        }
    }

    printf("%lld" , ans);

    return 0;

}

这道题还是很简单的，一开始我没有注意到每个宝盒必须取一个物品，写了滚动数组；没有注意到，凑成一个价值会有很多种方案（没有好好审题），计算答案时出错了
