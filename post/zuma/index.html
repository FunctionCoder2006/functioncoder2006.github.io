区间DP 动态规划

Zuma

Posted on 2021-12-28，2 min read

这道题告诉我们，很多题都是有关联的

首先，通过读题看样例我们大致感受一下：一个区间，既可以分成两个区间进行消除；也可以先消除一个区间的几个子区间，最后形成最后一个回文，把它消除掉

然后我们着重考虑第二种操作怎么做

我们发现，如果 al=ara_l = a_ral​=ar​ ，那么最后的结果可以等效成 fl+1,r−1f_{l + 1 , r - 1}fl+1,r−1​。因为我们考虑将 ala_lal​ 和 ara_rar​ 和其他数字形成最后一个回文进行消除，而根据回文的性质，两边删除或添加相同的数字，结果已然是回文

思路是哪里来的呢？其实就是 CQOICQOICQOI 涂色那道题，考虑 al=ara_l = a_ral​=ar​ 的情况

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cstring>

using namespace std;

const int N = 510;

int n , a[N] , f[N][N];

int main(void){

    cin >> n;

    for(int i = 1 ; i <= n ; ++ i){
        cin >> a[i];
    }

    memset(f , 0x3f , sizeof(f));

    for(int i = 1 ; i <= n ; ++ i){
        f[i][i] = 1;
    }

    for(int i = 1 ; i <= n ; ++ i){
        if(a[i] == a[i + 1]){
            f[i][i + 1] = 1;
        }
        else f[i][i + 1] = 2;
    }

    for(int Len = 3 ; Len <= n ; ++ Len){
        for(int l = 1 ; l + Len - 1 <= n ; ++ l){
            int r = l + Len - 1;
            if(a[l] == a[r])
                f[l][r] = min(f[l][r] , f[l + 1][r - 1]);
            for(int k = l ; k < r ; ++ k){
                f[l][r] = min(f[l][r] , f[l][k] + f[k + 1][r]);
            }
        }
    }

    printf("%d" , f[1][n]);

    return 0;

}

很多题目的操作和思想都是有一些相似和关联之处的，这就需要我们多多感受，多多思考
