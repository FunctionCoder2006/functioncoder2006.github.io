背包 动态规划

CSP-J 2019 纪念品

Posted on 2021-11-10，2 min read

Problem1Problem1Problem1：将 pi+1,j−pi,jp_{i + 1 , j} - p_{i , j}pi+1,j​−pi,j​ 作为价值。考虑对买卖方式进行转化。对于一个纪念品，第 jjj 天买入，第 iii 天卖出等价于 第 jjj 天买入，第 j+1j + 1j+1 天卖出，第 j+1j + 1j+1 天又买入，第 j+2j + 2j+2 天又卖出······因此我们不需要考虑当前手中持有多少个这个纪念品。还有一点就是我做了 t−1t - 1t−1 层循环，每一次虽然都将 fff 数组清零，看起来每一次 dpdpdp 没有任何联系，但是实际上，每做一次 dpdpdp 都会对 ansansans 进行一次更新， 更新了就意味着做了一轮交易（一共 ttt 轮）

Problem2Problem2Problem2：每一次将 ansansans 更新成最大值，因为我每一轮都在更新，所以 ansansans 就是经历了 iii 轮的最大利润，如果想让下一轮赚的更多的话就得有更多的资金

一个小细节：设 fif_ifi​ 表示投入 iii 价格的前的最大利润且最大利润是本轮的，不对其他轮构成影响。

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 1e2 + 10;
const int M = 1e4 + 10;

int t , n , m , f[M] , p[N][M] , ans;

int main(void){

    scanf("%d%d%d" , &t , &n , &m);

    for(int i = 1 ; i <= t ; ++ i)
        for(int j = 1 ; j <= n ; ++ j)
            scanf("%d" , &p[i][j]);

    ans = m;

    for(int i = 1 ; i < t ; ++ i){
        memset(f , 0 , sizeof(f));
        for(int j = 1 ; j <= n ; ++ j){
            for(int k = p[i][j] ; k <= ans ; ++ k){
                f[k] = max(f[k] , f[k - p[i][j]] + p[i + 1][j] - p[i][j]);
            }
        }
        ans = max(ans , f[ans] + ans);
    }

    printf("%d" , ans);

    return 0;
}
