枚举n的所有约数

inline void div(int x){
    for(int i = 1 ; i <= x / i ; ++i){
        if(x % i == 0){
            vec[x].push_back(i);
        }
    }
    int Size = vec[x].size();
    for(int i = Size - 1 ; i >= 0 ; --i){
        if(x / vec[x][i] != vec[x][i]){
            vec[x].push_back(x / vec[x][i]);
        }
    }
    return;
}

求出1~n所有约数

可以用倍数枚举法，就是先从小到大枚举一个iii，然后枚举iii的倍数jjj，那么iii一定是i∗ji * ji∗j的约数，而且因为iii是从小到大枚举的，所以是可以按照从小到大的顺序找到1 n1~n1 n的所有约数

for(int i = 1 ; i <= n ; ++i){
     for(int j = 1 ; j <= n / i ; ++j){
        vec[i * j].push_back(i);
    }
}
