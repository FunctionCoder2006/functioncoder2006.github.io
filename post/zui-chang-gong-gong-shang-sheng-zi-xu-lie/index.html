最长公共上升子序列

Posted on 2021-12-18，4 min read

当拿到这道题的时候，我的第一反应是最长公共子序列的推导过程

设 fi,jf_{i , j}fi,j​ 表示所有以 aia_iai​ 和 bjb_jbj​ 结尾的形成的公共上升子序列的方案中的最大长度

然后我们考虑 aaa 序列和 bbb 序列的倒数第二个数分别是哪一个，我们记做 kkk 和 jjj，因此，有

fi,j=max(fi,j,fk,j+1)，ai=bj，ak=bk,ak<aif_{i , j} = max(f_{i , j} , f_{k , j} + 1)，a_i = b_j，a_k = b_k , a_k < a_i fi,j​=max(fi,j​,fk,j​+1)，ai​=bj​，ak​=bk​,ak​<ai​

然后我们考虑前缀和优化。但是这个 ak<aia_k < a_iak​<ai​ 让我们很难受，因此，我们考虑枚举 kkk ，让 kkk 固定，设 gk,jg_{k , j}gk,j​ 表示最大的 fk,lf_{k , l}fk,l​ 其中 l∈[1,j]l ∈ [1 , j]l∈[1,j]，因此，有

fi,j=max(gk,j−1+1)，ak<ai,ai=bjf_{i , j} = max(g_{k , j - 1} + 1)，a_k < a_i , a_i = b_j fi,j​=max(gk,j−1​+1)，ak​<ai​,ai​=bj​

这里注意，gi,jg_{i , j}gi,j​ 并不代表我将 iii 固定，先枚举 jjj 并不影响正确性，这里思维定式了

然后考虑递推求 gi,jg_{i , j}gi,j​，有

gi,j=max(gi,j−1,max(gk,j−1)+1)，ai=bj，ak<aig_{i , j} = max(g_{i , j - 1} , max(g_{k , j - 1}) + 1)，a_i = b_j，a_k < a_i gi,j​=max(gi,j−1​,max(gk,j−1​)+1)，ai​=bj​，ak​<ai​

然后我们将 fi,jf_{i , j}fi,j​ 代换成 ggg 数组，因此，可以发现 ggg 数组的转移已经和 fff 数组无关了

考虑如何快速求 gk,j−1g_{k , j - 1}gk,j−1​，我们发现，如果我们先枚举 jjj 也就是让 jjj 固定对答案是没有影响的。因此，如果我们将 jjj 固定不动，让 iii 自增，我们的决策集合只会增加不会减少，因此，考虑继续使用前缀和优化

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 3e3 + 10;

int n , a[N] , b[N] , g[N][N];
int ans;

int main(void){

    scanf("%d" , &n);

    for(int i = 1 ; i <= n ; ++ i){
        scanf("%d" , &a[i]);
    }

    for(int i = 1 ; i <= n ; ++ i){
        scanf("%d" , &b[i]);
    }

    for(int j = 1 ; j <= n ; ++ j){
        int maxn = 0;
        for(int i = 1 ; i <= n ; ++ i){
            g[i][j] = g[i][j - 1];
            if(a[i] == b[j]){
                g[i][j] = max(g[i][j] , maxn + 1);
            }
            if(a[i] < b[j])
                maxn = max(maxn , g[i][j - 1]);
        }
    } 

    for(int i = 1 ; i <= n ; ++ i){
        ans = max(ans , g[i][n]);
    }

    printf("%d" , ans);

    return 0;

}

我们这么想，我们的 jjj 是固定的，在 iii 不断自增的时候，如果我此时发现 ai=bja_i = b_jai​=bj​ 说明gi,jg_{i , j}gi,j​ 可以尝试被更新了，而此时 ai=bja_i = b_jai​=bj​ ，因此，不管 iii 在哪儿，只要可以尝试被更新，就说明 ai=bja_i = b_jai​=bj​ ，就需要让 maxnmaxnmaxn 满足 ai<bja_i < b_jai​<bj​

其实，这句话就是让尝试将 gi,j−1g_{i , j - 1}gi,j−1​ 加入 gi+1,jg_{i + 1 , j}gi+1,j​ 决策集合，尝试着更新 gi+1,jg_{i + 1 , j}gi+1,j​
