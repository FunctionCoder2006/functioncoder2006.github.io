尼克的任务

Posted on 2021-12-18，3 min read

这道题总的来说是一道好题，感觉至今仍然没有吸收完全

根据之前的经验，我们考虑设 fif_ifi​ 表示所有时刻点到达 iii 的方案中的最大空暇时间，有

fi=max(fi,fi−tj)f_i = max(f_i , f_{i - t_j}) fi​=max(fi​,fi−tj​​)

但是我们很快发现这么转移是错的，有可能 fi−tjf_{i - t_j}fi−tj​​ 是非法的，有可能我们压根儿就不能做这个工作

所以，马上转换思路

其中一种挺妙的，考虑改变 fff 数组的含义，设 fif_ifi​ 表示所有从 iii 开始一直到 nnn 的工作方案中能获得的最大空暇时间，然后倒着求 fff 数组，有

fi=max(fi,fi+1+1),cnti=0f_i = max(f_i , f_{i + 1} + 1) ,cnt_i = 0 fi​=max(fi​,fi+1​+1),cnti​=0

fi=max(fi,fi+tj)，cnti≠0f_i = max(f_i , f_{i + t_j})，cnt_i ≠ 0 fi​=max(fi​,fi+tj​​)，cnti​​=0

其中 cnticnt_icnti​ 表示 iii 时刻任务的个数

这样的话，我们就不需要考虑 fi+tjf_{i + t_j}fi+tj​​ 是否合法了

另一种思路是通过设置负无穷来排除非法状态的干扰

但是我们需要使用填表法因为刷表法没有办法使用

#include<iostream>
#include<cstdio>
#include<cmath>
#include<cstring>
#include<cstring>
#include<vector>

using namespace std;

const int N = 1e4 + 10;

int n , k , f[N];

vector<vector<int> > vec(N);

int main(void){

    cin >> n >> k;

    memset(f , ~0x3f , sizeof(f));

    int p , t;

    for(int i = 1 ; i <= k ; ++ i){
        cin >> p >> t;
        vec[p].push_back(t);
    }

    f[1] = 0;

    for(int i = 1 ; i <= n ; ++ i){
        if(vec[i].size() == 0){
            f[i + 1] = max(f[i + 1] , f[i] + 1);
        }
        else {
            for(int j = 0 ; j < vec[i].size() ; ++ j){
                f[i + vec[i][j]] = max(f[i + vec[i][j]] , f[i]);
            }
        }
    }

    cout << f[n + 1] << endl;

    return 0;

}

其实，这道题记忆化搜索是最简单的，因为我们可以直接跳过非法状态
