配对统计

Posted on 2021-12-23，6 min read

这是一道好题，通过这道题，收获了不少

首先，通过观察题面，我们可以发现，对于一个 axa_xax​，一定是和它值相差最小的会为好的配对（当时还以为与它值最相邻的两个可以成为）

然后我们很自然的想到将 aia_iai​ 排序

然后发现，我们可以将两个点对抽象成一个区间，那么第 iii 个区间所有的点对就是点对区间在 [l,r][l , r][l,r] 的个数

根据 HHHHHH 的项链那道题的经验，我们可以将题目中给定的区间按照右端点排序，然后使用前缀和相减即 query(range[i].r)−query(range[i].l−1)query(range[i].r) - query(range[i].l - 1)query(range[i].r)−query(range[i].l−1)

然后我就想到了双指针，维护一个指针 jjj ，保持点对区间的右端点小于等于当前查询区间的右端点

然后我们发现，当前枚举的所有的点对区间已经全部小于等于查询区间右端点（前提是点对区间已经按照右端点排好序了）,我们只需要知道所有左端点在 [range[i].l,range[i].r][range[i].l , range[i].r][range[i].l,range[i].r] 的点对区间的个数就可以了

不管点对区间的右端点在哪里，只要它的左端点没有在 [range[i].l,range[i].r][range[i].l , range[i].r][range[i].l,range[i].r]，那么，它一定是不合法的

因此我们只需要用树状数组维护一个前缀和 c[x]c[x]c[x] 表示 [1,x][1 , x][1,x] 左端点个数的前缀和就可以了

值得一提的是，这道题不能直接用前缀和，因为我每一个区间都要查询一次。如果扫描完区间后再查询就完了，这时候点对区间的右端点可能就不满足要求了

还有一点需要注意，(x,y)(x , y)(x,y) 点对是合法的，不意味着 (y,x)(y , x)(y,x) 也是合法的。区别就是一个是以 xxx 为基准的，一个是以 yyy 为基准的，两者是不一样的

这道题给我们的启发就是题目与题目之间很多操作、思想都是有联系的，这就需要我们平时多多总结，加深对这个解题思想和操作的印象

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 3e5 + 10;

int n , m , a[N] , cnt , idx , c[N];
long long ans;

pair<int , int> s[N];
pair<int , int> b[N];

struct node{
    int l , r , id;
    bool operator < (const node &x)const{
        return r < x.r;
    }
    node(){}
    node(int _l , int _r , int _id){
        l = _l , r = _r , id = _id;
    }
}range[N];

inline int lowbit(int x){
    return x & (-x);
}

inline void update(int x , int v){
    for(int i = x ; i < N ; i += lowbit(i)){
        c[i] += v;
    }
    return;
}

inline int query(int x){
    int s = 0;
    for(int i = x ; i >= 1 ; i -= lowbit(i)){
        s += c[i];
    }
    return s;
}

bool cmp(pair<int , int> x , pair<int , int> y){
    if(x.second == y.second)
        return x.first < y.first;
    return x.second < y.second;
}

int main(void){

    ios::sync_with_stdio(false); cin.tie(0); cout.tie(0);

    cin >> n >> m;

    if(n == 1){
        cout << 0 << endl;
        return 0;
    }
    for(int i = 1 ; i <= n ; ++ i){
        cin >> a[i];
        s[i] = make_pair(a[i] , i);
    }

    sort(s + 1 , s + 1 + n);

    for(int i = 1 ; i <= m ; ++ i){
        cin >> range[i].l >> range[i].r;
        range[i].id = i;
    }

    sort(range + 1 , range + 1 + m);

    b[++ cnt] = make_pair(min(s[1].second , s[2].second) , max(s[1].second , s[2].second));
    b[++ cnt] = make_pair(min(s[n].second , s[n - 1].second) , max(s[n].second , s[n - 1].second));

    for(int i = 2 ; i < n ; ++ i){
        if(s[i].first - s[i - 1].first <= s[i + 1].first - s[i].first){
            b[++ cnt] = make_pair(min(s[i].second , s[i - 1].second) , max(s[i].second , s[i - 1].second));
        }
        if(s[i + 1].first - s[i].first <= s[i].first - s[i - 1].first){
            b[++ cnt] = make_pair(min(s[i].second , s[i + 1].second) , max(s[i].second , s[i + 1].second));
        }   
    }

    sort(b + 1 , b + 1 + cnt , cmp);
    
    int j = 1;

    for(int i = 1 ; i <= m ; ++ i){
        while(j <= cnt && b[j].second <= range[i].r){
            update(b[j].first , 1);
            ++ j;
        }
        ans += 1ll * (query(range[i].r) - query(range[i].l - 1)) * range[i].id;
    }
    cout << ans << endl;
    
    return 0;

}
