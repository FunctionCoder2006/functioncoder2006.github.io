归并排序求逆序对

Posted on 2021-12-29，2 min read

首先我们先介绍一下归并排序

其实非常简单，将一个区间分成两个区间 [l,mid][l , mid][l,mid]，[mid+1,r][mid + 1 , r][mid+1,r]，将这两个区间进行递归排序，然后尝试着将 这两个区间进行序列合并，合并成有序的序列

然后问我们考虑怎么求逆序对

对于一个区间 [l,r][l , r][l,r] ，我们可以将其分为两半，然后分别算出左半边逆序对个数和右半边逆序对个数，和跨过 midmidmid 分界线的逆序对数

左右半边很好处理，直接递归下去解决就可以

我们考虑跨过 midmidmid 的逆序对数怎么求

因为我们考虑的是跨过 midmidmid 的逆序对，因此与两半边各自的顺序无关（在左边就是在左边，在右边就是在右边）。此时，我们不妨考虑将每一半边都进行升序排序（这个归并排序顺便就可以完成，不需要消耗额外的复杂度），然后使用双指针算法

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 5e5 + 10;

int n , a[N] , b[N];
long long ans;

inline void merge(int l , int r){
    if(l == r)return;
    int mid = (l + r) >> 1;
    merge(l , mid); merge(mid + 1 , r);
    int i = l , j = mid + 1;
    for(int k = l ; k <= r ; ++ k){
        if(j > r || (i <= mid && a[i] <= a[j]))
            b[k] = a[i] , ++ i;
        else b[k] = a[j] , ++ j , ans += (mid - i + 1);
    }
    for(int k = l ; k <= r ; ++ k){
        a[k] = b[k];
    }
    return;
}

int main(void){
    
    scanf("%d" , &n);
    
    for(int i = 1 ; i <= n ; ++ i){
        scanf("%d" , &a[i]);
    }
    
    merge(1 , n);
    
    printf("%lld" , ans);
    
    return 0;
    
}
