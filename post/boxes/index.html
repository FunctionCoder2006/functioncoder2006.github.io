顺着题意，考虑从上往下放箱子。

设 fi,jf_{i , j}fi,j​ 表示所有只考虑前 iii 个箱子，已经放了重量为 jjj的方案中最多能放多少箱子。

对于第 iii 个箱子，显然有放和不放两种情况。

不放这种情况很简单，就是 fi−1,jf_{i - 1 , j}fi−1,j​。

如果放的话我们需要考虑前 i−1i - 1i−1 个箱子所放的重量是否超过第 iii 个箱子的最大载重，因此我们需要保证 di≥j−wid_i ≥ j - w_idi​≥j−wi​ did_idi​ 表示第 iii 个箱子的最大载重，wiw_iwi​ 表示第 iii 个箱子的重量。因此，如果选第 iii 个箱子，有 fi,j=max(fi−1,j,fi−1,j−wi+1)f_{i , j} = max(f_{i - 1 , j} , f_{i - 1 , j - w_i} + 1)fi,j​=max(fi−1,j​,fi−1,j−wi​​+1)

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 3e5 + 10;

int w[N] , d[N] , n , f[N];

int main(void){

    ios::sync_with_stdio(false); cin.tie(0); cout.tie(0);

    while(cin >> n){

        if(n == 0)break;
               
        for(int i = 1 ; i <= n ; ++ i){
            cin >> w[i] >> d[i];
        }

        memset(f , 0 , sizeof(f));

        for(int i = n ; i >= 1 ; -- i){
            for(int j = 6000 ; j >= w[i] ; -- j){
                if(d[i] >= j - w[i])f[j] = max(f[j] , f[j - w[i]] + 1);
            }
        }

        int ans = 0;

        for(int i = 0 ; i <= 6000 ; ++ i){
            ans = max(ans , f[i]);
        }

        cout << ans << endl;

    }
    return 0;

}

还有一个点需要注意，对于一个箱子，它的最大载重是 3000 ，它的重量最大也是 3000，那么我们能放的最大重量是 6000。
