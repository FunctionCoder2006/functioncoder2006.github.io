埃氏筛

为了学区间筛还得被迫学一下埃氏筛……
埃氏筛就是让一个素数（不一定是最小质因子）筛掉一个合数。其中有一个优化，设一个数kxkxkx其中kkk为正整数且k∈[1,x)k∈[1 , x)k∈[1,x)自然kx＜x2kx ＜ x^2kx＜x2因为k<xk < xk<x所以比xxx小的xxx的倍数一定会被某一个kkk筛掉吗，所以我们直接从xxx倍数开始筛

inline void Prime(int n){
    for(int i = 2 ; i <= n ; ++i){
        if(v[i])continue;//用一个质数去筛掉一个合数因为合数一定会有一个质因子
        for(int j = i ; j <= n / i ; ++j){//j从i开始枚举就意味着i * j >= x * x
            v[i * j] = true;
        }
    }
    return;
}

线性筛

线性筛就是让每一个合数都没ta的最小质因子筛掉，这样只会保证每个合数只会被筛掉一次，具体细节看代码……

inline void Prime(int n){
    int cnt = 0;
    for(int i = 2 ; i <= n ; ++i){
        if(!mindiv[i]){
            mindiv[i] = i;
            prime[++ cnt] = i;
        }
        for(int j = 1 ; j <= cnt ; ++j){
            if(prime[j] > n / i || prime[j] > mindiv[i])
                break;//因为prime质数表里是单调递增的，所以当prime[j] > i的话prime[j]就不是i * prime[j]的最小质因子了
            mindiv[i * prime[j]] = prime[j];
            if(i % prime[j] == 0)
                break;//程序度完这一句prime[j]就会变大，如果发现prime[j]是i的因子并且prime[j]在下一次循环时就会变大，所以变大了的prime[j]一定不是i * prime[j]的最小质因子（因为i有比变大后prime[j]更小的质因子也就是没有变的的prime[j]）
        }
    }
    return;
}
