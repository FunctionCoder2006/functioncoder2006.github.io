杨老师的照相排列

2021-12-08 #线性dp #动态规划

这道题被自己浪费掉了……

首先有一点，k≤5k ≤ 5k≤5 我以为这就是数据水，殊不知这是题目给我的提示；其次，没有留意 N1≥N2≥N3≥N4≥N5N1 ≥ N2 ≥ N3 ≥ N4 ≥ N5N1≥N2≥N3≥N4≥N5 。这道题一直在想之前练的区间划分的做法，但是很显然没有办法转移。

接下来就是正解

首先，当我们看到 k≤5k ≤ 5k≤5，n≤30n ≤ 30n≤30 的时候，应该大胆将设五维状态（写到这，我又想到了 NOIPNOIPNOIP 仿真模拟）。因此，设 fa,b,c,d,ef_{a , b , c , d , e}fa,b,c,d,e​ 表示第一排已经站了 aaa 个人，第二排 bbb 个人，以此类推

然后我们思考如何转移，题目中要求每一排每一列单调递减很让我们头疼。但是我们可以考虑将 iii 号同学放到哪个位置，因为在我们枚举 iii 的时候，此时 iii 就是最大的。

但是我们不是将 iii 随意放置，我们必须保证 iii 放入这一排以后这排的人数比之前排的人数要小。为什么呢？因为我们是将 111 ~ nnn 逐个放置，如果 iii 前面有空位置，那么在枚举到其他数的时候就会把这个位置填上，而这个数一定要大于 iii ，不符合题设

然后我们还会发现一个问题，题解里面并没有枚举 iii 也就是第 iii 个人，为什么呢？因为我枚举了每一排有多少人，因此我是一定可以确定 iii 是多少，所以 iii 是一个冗余状态，不需要枚举

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 31;

int n , s[N];
 long long f[N][N][N][N][N];

int main(void){

    while(1){

        cin >> n;

        if(n == 0)break;

        memset(s , 0 , sizeof(s));

        for(int i = 1 ; i <= n ; ++ i){
            scanf("%d" , &s[i]);
        }

        memset(f , 0 , sizeof(f));

        f[0][0][0][0][0] = 1;

        for(int a = 0 ; a <= s[1] ; ++ a)
            for(int b = 0 ; b <= min(a , s[2]) ; ++ b)
                for(int c = 0 ; c <= min(b , s[3]) ; ++ c)
                    for(int d = 0 ; d <= min(c , s[4]) ; ++ d)
                        for(int p = 0 ; p <= min(d , s[5]) ; ++ p){
                            if(a + 1 <= s[1])f[a + 1][b][c][d][p] += f[a][b][c][d][p];
                            if(b + 1 <= a && b + 1 <= s[2])f[a][b + 1][c][d][p] += f[a][b][c][d][p];
                            if(c + 1 <= b && c + 1 <= s[3])f[a][b][c + 1][d][p] += f[a][b][c][d][p];
                            if(d + 1 <= c && d + 1 <= s[4])f[a][b][c][d + 1][p] += f[a][b][c][d][p];
                            if(p + 1 <= d && p + 1 <= s[5])f[a][b][c][d][p + 1] += f[a][b][c][d][p];
                        }

        printf("%lld\n" , f[s[1]][s[2]][s[3]][s[4]][s[5]]);


    }

    return 0;

}

这里有两点需要注意，一点是每次都要将 sss 数组清空，否则上一次的 sss 有可能会影响到这一次的；还有一点是我们使用 printfprintfprintf 输出 longlonglonglonglonglong 的时候要写成 lldlldlld
