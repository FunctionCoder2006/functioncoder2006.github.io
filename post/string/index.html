区间DP 动态规划

String

Posted on 2021-12-26，4 min read

这道题是一道好题，通过这道题收获了不少

首先，通过读题，大致能感受出来，我们一定是将若干个连续的并且互不相交的区间进行粉刷。话句话说，就是我不管怎么粉刷，一定会出现一些区间被粉刷，一些区间没有被粉刷

那么我们可以尝试着用区间划分的方法来枚举这些区间的所有情况

设 fif_ifi​ 表示所有将 AAA 的 [1,i][1 , i][1,i] 粉刷成 BBB 的 [1,i][1 , i][1,i] 的方案

那么此时就要分两种情况了，一种是粉刷 iii ，另一种是不粉刷 iii

  • 如果粉刷 iii ，我们考虑将 [j+1,i][j + 1 , i][j+1,i] 这一段区间进行一次粉刷，有

    fi=max(fi,fj+gj+1,i)，j∈[1,i−1]f_i = max(f_i , f_j + g_{j + 1 , i})，j∈[1 , i - 1] fi​=max(fi​,fj​+gj+1,i​)，j∈[1,i−1]

  • 如果不粉刷 iii，此时的结果就相当于 fi−1f_{i - 1}fi−1​ 也就是我对 i−1i - 1i−1 考虑要不要进行粉刷

对于第一种情况，在进行区间 dpdpdp 的时候是不用考虑不粉刷的

因为你考虑我本来就是在进行区间划分，本来就是在枚举最后一段粉刷哪一段区间，因此不需要考虑不粉刷 （就算算的不对算大了又怎样） ，如果不想进行粉刷我们不是还有 fi−1f_{i - 1}fi−1​ 嘛。

好好体会一下，这里的 fi−1f_{i - 1}fi−1​ 也可以形成一段连续的区间只是这个区间非常简单我们没有进行任何操作

这告诉我们如果一个区间什么都没有做我们可以考虑 fi−1f_{i - 1}fi−1​ ；我们在进行 dpdpdp 的时候一定要知道自己是来干什么的，不要拘泥于死板的 dpdpdp 方程。比如这道题，我们一定要清楚自己是来枚举最后一段可能粉刷的区间不需要考虑不粉刷

另一种原因是如果我考虑不粉刷的话，那我每一个状态都需要考虑不粉刷，但是最优解中大概率存在粉刷一段连续的区间，而此时这个区间不能考虑不粉刷。所以，如果我考虑不粉刷的话，不满足子问题的相似性

#include<iostream>
#include<cstdio>
#include<algorithm>
#include<cstring>
#include<cmath>

using namespace std;

const int N = 510;

char a[N] , b[N];
int g[N][N] , f[N];

int main(void){

    while(scanf("%s%s" , a + 1 , b + 1) != EOF){
        int n = strlen(a + 1) , m = strlen(b + 1);
        memset(g , 0x3f , sizeof(g));
        for(int i = 1 ; i <= n ; ++ i){
            g[i][i] = 1;
        }
        for(int Len = 2 ; Len <= n ; ++ Len){
            for(int l = 1 ; l + Len - 1 <= n ; ++ l){
                int r = l + Len - 1;
                if(b[l] == b[r]){
                    g[l][r] = min(g[l][r - 1] , g[l + 1][r]);
                }
                for(int k = l ; k < r ; ++ k){
                    g[l][r] = min(g[l][r] , g[l][k] + g[k + 1][r]);
                }
            }
        }
        memset(f , 0x3f , sizeof(f));
        f[0] = 0;
        for(int i = 1 ; i <= n ; ++ i){
            if(a[i] == b[i]){
                f[i] = f[i - 1];
            }
            f[i] = min(f[i] , g[1][i]);
            for(int j = 1 ; j < i ; ++ j){
                f[i] = min(f[i] , f[j] + g[j + 1][i]);
            }
        }
        printf("%d\n" , f[n]);
    }

    return 0;

}
